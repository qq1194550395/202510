# 数据集转换工具功能列表

## 🔄 核心转换功能

### 支持的格式
- **YOLO检测格式**: 矩形框标注 (class cx cy w h)
- **YOLO分割格式**: 多边形标注 + 矩形框混合
- **VOC格式**: XML标注文件
- **JSON格式**: 自定义JSON标注格式

### 转换路径
- YOLO检测 ↔ VOC
- YOLO检测 ↔ JSON
- YOLO分割 ↔ JSON
- YOLO分割 → YOLO检测 (只保留矩形框)
- JSON → YOLO分割

## 📊 数据集分析功能

### 统计分析
- **基本统计**: 图片数量、标注数量
- **类别分布**: 各类别标注数量统计
- **图片尺寸分析**: 平均尺寸、尺寸范围
- **标注密度**: 每张图片平均标注数
- **数据平衡性**: 类别分布均衡度分析

### 质量检查
- **文件完整性**: 检查图片和标注文件匹配
- **格式验证**: 验证标注文件格式正确性
- **坐标范围检查**: 确保坐标在有效范围内
- **健康度评分**: 0-100分的数据集质量评分
- **问题报告**: 详细的问题列表和修复建议

## 🔧 数据集整理功能

### 文件管理
- **批量重命名**: 统一文件命名格式

- **数据集合并**: 合并多个数据集
